# Grupo05_LDP-GT3
Venta de Boletos de Cine

Una Película se proyecta en diferentes Salas De Cine en
distintas Funciones (horarios). Un Cliente puede comprar uno o más boletos para
una función específica. Se debe controlar el número de asientos disponibles para
cada función para evitar la sobreventa.Control estricto de asientos específicos para
evitar vender la misma silla a dos clientes.

## Stack Tecnológico

Este proyecto utiliza las siguientes herramientas:

- **Backend:** Python con el framework Django.
- **Frontend:** Tailwind CSS para los estilos y componentes.
- **Gestión de Dependencias:** uv.
- **Calidad de Código:** Ruff (Linter) y Mypy (Type checking).
